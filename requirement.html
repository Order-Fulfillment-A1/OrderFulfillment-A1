Carrier Appointment

User
  • Shipment Planner
    Liked the UI with a view that doesn't give much distraction to make it easier to arrange shipping routes and which trackpoints will be visited during shipping. And liked the simple UI display design with color contrast between windows to make it easier to do two different roles.
  • Junior Supply Officer
    Liked the organized UI with contrast between one button and another to differentiate the work window as it needed to do some work.
  • Senior Supply Officer
    Liked the UI with a comfortable display for reading large numbers of documents. And liked the simple UI display design with color contrast between windows to make it easier to do two different roles.
  • Warehouse Office
    Liked the UI with a calendar view with the availability of many color choices to provide notes on each date the package was picked up to be sent. As well as a display design that makes it easy to see which packages need to come first.
  • Warehouse Admin Officer
    Liked the UI with a simple interface and easy to do progress notification related to other roles.
Task Description
  • Create Route Guide and Estimate Trailer Usage
    After the Purchase Order is confirmed, the Shipment Planner will create a route guide and trailer usage. The route guide is a document that contains which trackpoints must be passed when the delivery takes place. While trailer usage is a document that contains how much trailer needs are needed by estimating the size of the order.
  • Prepare Transportation Quote
    If the Route Guide and Trailer Usage are completed on time by the Shipment Planner, the Junior Supply Officer will carry out a Prepare Transportation Quote by calculating the shipping costs based on the number of packages sent and the total volume of fraight shipped. The results of this calculation will be divided into 3 (Single Package, Less than Truckload, Full Truck-Load).
  • Single Package
    If the package sent is as a single package, then the entire process will then be handled by the Junior Supply Officer, by making a Motor Carrier Pickup Manifest.
  • Less than Truckload
    If the package sent is Less than Truckload, then the Warehouse Officer will then make a Pickup and Delivery Appointment. After that, Senior Supply Officer will make a Bill of Lading document. If the Warehouse Officer does not make a Delivery Appointment, the Delivery Appointment will be made by the Junior Supply Officer.
  • Full Truck-Load
    If the package sent is Full Truck-Load, the Delivery Appointment and Pickup Appointment can be made by the Client Liaison or Warehouse Officer & Junior Supply Officer. Then, the Senior Supply Officer will create a Shipment Information Document.
  • Produce Shipment Notice
    This Shipment Notice will only be made if the delivery category is less than truckload and full truckload. Shipment Notice is made by the Warehouse Admin Officer.

Freight In Transit

User
  • Courier
    Likes the UI with a simple look with the use of icons that are easy to used to make it easier for Courier to record track points. Also, a neat display to choose which job to do first.
  • Carrier Admin Officers
    Likes the UI with a simple look. Carrier admin officers also need a UI with an organized design to make their work easier, as well as a display design that makes it easier to see which work needs to come first, or change tasks with other Carrier admin officers.
  • Client Liaison
    Likes the UI with a simple and well-organized interface to make work easier, as well as a display design that made it easy to reply to questions asked by customers.
  • PO Manager
    Likes the organized UI and the contrast between one button and another to differentiate the work window as it needed to do some of the work that was already available. Also, a simple UI for creating acceptance certificates.
Task Description
  • The Freight in Transit sub-process is carried out in parallel with the Payment sub-process.
  • A delivery truck may visit several destinations and they may designate some of their track points.
  • The Courier, which records the arrival and departure times of the trucks, issues track point notifications.
  • Carrier Admin Officers need to record a Trackpoint Order Entry for each track point, which contains a report for that Trackpoint.
  • They can log order entries in parallel as long as Carrier Admin Officers are available to work on the entries.
  • While entries are being recorded, customer inquiries need to be handled by the Client Liaison.
  • After the transit is complete, the PO Manager creates an Acceptance Certificate, which shows that they have physically sent the shipment.

Freight Delivered

User
  • Client Liaison
    Likes the UI with a simple and well-organized interface to make work easier, as well as a display design that made it easy to reply to questions asked by customers.
  • Senior Supply Officer
    Likes the UI with a comfortable appearance for reading large numbers of documents. As well as a display design that makes it easy to see which work needs to come first.
Task Description
  • After delivery of the goods, the client may apply for a Return of Goods or may wish to file a Loss or Damage Claim.
  • This Return Document, or Loss or Damage Claim, is entered into the system by the Client Liaison at the request of the client.
  • If they receive no client communication within a certain period after delivery, the Order Fulfillment process is complete.
  • The Senior Supply Officer must approve requests for returns.
  • If approved, process on Return Management. The same applies to claims for loss or damage.
